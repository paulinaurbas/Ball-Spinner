In the Car folder you ll find a few ready made Car which are used in the Car sample scenes These prefabs demonstrate how you can use the Car scripts we ve including some AI controlled variants which are able to drive directly towards a specified or follow a waypoint based route The scripts included have been designed to be modular in so that the extra features of a particular car can be set up (such as brake lights, engine sound, and skid trails). The simplest way to get started setting up your own car is to use our "CarPrefabEmpty"
